# Interface Utilisateur

Ce dossier contient tout le code frontend de l'application MakerLens.

## Contenu

- **`index.html`** : La page d'accueil de l'application.
- **`a_propos.html`** : La page "À propos".
- **`css/`** : Les feuilles de style (styles.css).
- **`js/`** : Les scripts JavaScript (app.js).
- **`assets/images/`** : Les images utilisées dans l'interface.

## Développement

Pour modifier l'interface, éditez les fichiers HTML, CSS ou JS. Ouvrez `index.html` dans un navigateur pour voir les changements.
