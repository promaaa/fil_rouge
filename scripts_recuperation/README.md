# Scripts de Récupération

Ce dossier contient les outils pour récupérer (scraper) et analyser les données depuis le web.

## Structure

- **`scripts/`** : Scripts principaux de scraping (anciennement `scrapers`).
- **`analyseurs/`** : Scripts d'analyse et de parsing des pages (anciennement `parsers`).
- **`donnees_brutes/`** : Données JSON brutes extraites (anciennement `json`).
- **`pages_brutes/`** : Pages HTML brutes téléchargées (anciennement `pages_html`).
- **`utilitaires/`** : Fonctions utilitaires partagées.

## Utilisation

Consultez les scripts individuels pour les instructions d'exécution.
